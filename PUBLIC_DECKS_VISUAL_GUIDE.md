## Responsive Design

### Desktop (>= 1024px)
- Full navigation bar with all tabs
- 3-column grid for deck cards
- Wide modals with side-by-side layouts
- Search bar visible in header

### Tablet (768px - 1023px)
- Compact navigation with abbreviated labels
- 2-column grid for deck cards
- Horizontal scroll for navigation if needed
- Modals adapted to medium width

### Mobile (< 768px)
- Hamburger menu for navigation
- Single column for deck cards
- Full-width modals
